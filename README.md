# PROYECTO-GESTION-ACADEMICA
Descripción del proyecto

Este proyecto consiste en el desarrollo de un Sistema de Gestión Académica implementado en Java. Su objetivo es facilitar la administración y organización de las materias mediante el uso de estructuras de datos como Árboles Binarios de Búsqueda (ABB) y Grafos.

El sistema permite registrar, buscar y visualizar información relacionada con las materias, además de gestionar las relaciones existentes entre ellas. Para ello se emplean algoritmos y estructuras de datos que garantizan una gestión eficiente de la información.

Funcionalidades principales

- Registrar materias en el sistema.
- Buscar materias por diferentes criterios.
- Mostrar las materias almacenadas.
- Gestionar relaciones entre materias mediante grafos.
- Organizar la información utilizando Árboles Binarios de Búsqueda.

Estructura del proyecto

El proyecto está compuesto por las siguientes clases:

- Materia: almacena la información de cada materia.
- NodoABB: representa un nodo del árbol binario de búsqueda.
- ArbolABB: implementa las operaciones del árbol binario de búsqueda.
- Grafo: gestiona las relaciones entre las materias.
- SistemaAcademico: coordina las funcionalidades principales del sistema.
- Main: contiene el método principal para la ejecución de la aplicación.

Tecnologías utilizadas

- Java
- IntelliJ IDEA
- Programación Orientada a Objetos (POO)
